# DeepSeek Coder V2

## Overview

DeepSeek Coder V2 (and V3) is an open-weight coding model from DeepSeek, currently one of the best open-weight coding models available. It runs locally via Ollama and provides capable AI coding assistance at zero cost.

## Capabilities

- Code generation and completion
- Multi-language support including Python, C++, C#, and scripting languages
- Can assist with Godot/GDScript development
- Runs on consumer hardware with quantized models

## Integration

Used with:
- **Ollama** -- local model runner
- **Continue** -- IDE extension for AI coding assistance
- Any editor supporting OpenAI-compatible API

## Cost

Free and open-source -- no API costs, runs entirely on local hardware.
