# Paradox Development Studio

## Overview

- Subsidiary of Paradox Interactive
- **Genre speciality**: Grand strategy games
- **Key title**: [[Crusader Kings III]] (2020) — the definitive feudal simulation game
- **Approach**: Decades of accumulated strategy game development legacy (CK1 + CK2 lineage)

## Design Significance

CK3's vassal personality system, faction dynamics, and approach to succession as core gameplay represent the gold standard for feudal game mechanics. However, the overwhelming complexity and DLC-heavy development model serve as cautionary examples.
