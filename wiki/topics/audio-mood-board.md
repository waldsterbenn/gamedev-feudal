# Audio Mood Board

## Overview

The Audio Mood Board captures the sonic identity of the game, defining what the game should sound like across music, sound effects, and voice/dialogue channels.

## Music Direction

Music style should address instrumentation and genre:

| Direction | Description |
|-----------|-------------|
| **Medieval/Folk Authentic** | Period instruments: lute, hurdy-gurdy, bagpipes |
| **Cinematic Orchestral** | Full orchestra, dramatic, modern feeling |
| **Hybrid** | Period instruments with modern production techniques |
| **Minimal Ambient** | Drone, tension, atmosphere-focused |

Reference tracks should identify what to borrow: tempo, instrumentation, mood.

## Sound Effects Style

| Category | Design Focus |
|----------|-------------|
| **Combat** | Weight, impact, realism vs stylized |
| **UI Feedback** | Subtle clicks or pronounced sounds |
| **Ambient** | Density, layering, environmental presence |
| **Foley** | Footsteps, cloth movement, equipment sounds |

## Voice and Dialogue

The audio mood board should define the approach to vocal content:
- Full voice acting with specific style
- Text-only with portrait expressions
- Narrated cutscenes only

## Implementation

Audio references should follow the same system as visual references:
- Store reference audio files in the project assets
- Document what specifically is being referenced
- Link to the [[Audio Design]] document for technical implementation
