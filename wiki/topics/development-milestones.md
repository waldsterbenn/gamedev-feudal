# Development Milestones

The project follows a phased milestone model from concept through launch. Each milestone has a target date, status, and specific deliverables.

## Milestone Phases

| Milestone | Target Date | Status | Notes |
|-----------|-------------|--------|-------|
| **Concept & Pre-Production** | 2026-Q2 | In Progress | Core docs, prototypes |
| **Vertical Slice** | TBD | Not Started | One complete gameplay loop |
| **Alpha** | TBD | Not Started | All features, placeholder art |
| **Beta** | TBD | Not Started | Polished, all content |
| **Release Candidate** | TBD | Not Started | Bug fixes, optimization |
| **Launch** | TBD | Not Started | Public release |

## Key Deliverables by Phase

### Concept & Pre-Production
- Core documentation
- Early prototypes
- Initial [[Sprint Planning]] setup

### Vertical Slice
- One complete [[Design Decisions|gameplay loop]] with final or near-final quality
- Proves art style, UI flow, and core fun
- 60-90 seconds of polished gameplay

### Alpha
- All features implemented with placeholder art
- [[Team Structure|Full team]] collaboration on content integration

### Beta
- All content finalized
- Polished to near-release quality
- [[QA Strategy|intensive quality assurance]] begins

### Release Candidate
- Bug fixes and performance optimization
- [[Bug Tracking|Known issues]] resolved
- Target platform [[Testing Plan|build testing]] complete

### Launch
- Public release on target platforms
- Post-launch monitoring and [[Playtesting Framework|feedback collection]]
