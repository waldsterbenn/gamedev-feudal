# Raw Sources

This folder holds immutable source documents that the LLM reads but never modifies.

## How to add sources

1. Drop any document here (markdown, text, or converted articles)
2. Tell the LLM to ingest it: "Ingest raw/filename.md"
3. The LLM will read it, update the wiki, update index.md, and append to log.md
