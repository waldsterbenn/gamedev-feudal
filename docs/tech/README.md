# Gamedev-Feudal Technical Documentation

This directory contains the central technical standards and developer instructions for the `gamedev-feudal` project.

## Core Documentation
- **[Developer Instruction](developer-instruction.md)**: The primary reference for technical specifications, architectural standards, and workflow rules governing all development in this project.
- **[Agent Manifest](../src/AGENTS.md)**: The self-contained instruction set for automated coding agents, outlining iteration loops, tool preferences, and adherence requirements.

## Development Standards
All contributors and automated agents must adhere to the patterns defined in the **[Developer Instruction](developer-instruction.md)** to ensure project maintainability, adherence to the component-based pattern, and efficient 3D spatial management.
