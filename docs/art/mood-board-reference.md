# Visual & Audio Reference Guide (Mood Board)

> **Purpose:** Collect and organize reference material for artists, audio designers, and the whole team to maintain a consistent feel.
> **Rule:** Before creating any art or audio, check this document first.

---

## Why This Exists

"Make it look medieval" means ten different things to ten different people. One person sees Dark Souls, another sees Disney, another sees Crusader Kings, another sees a pixel-art roguelike.

This document aligns everyone on **what the game should look and sound like** before any work is invested in the wrong direction.

---

## How to Use This

1. Add references **as images** (screenshots, concept art, photos) in `assets/art/reference/`
2. Link them below with a brief note about **what specifically you're referencing**
3. Be specific: don't just say "like Dark Souls" -- say "the lighting and color palette of Firelink Shrine in Dark Souls"
4. Update this whenever new references are discovered or the visual direction evolves

## Collecting New References

### How to Add a Reference

1. Save the image file in `assets/art/reference/` with a clear name: `ref_firelink-shrine-lighting.jpg`
2. Add it to the appropriate table above
3. Write 1-2 sentences about **exactly what** you're referencing and **why**
4. Note which doc it influences (art-style-guide.md, audio-design.md, etc.)

### Where to Find Good References

- **Pinterest** -- create a shared board for the team
- **ArtStation** -- professional game art references
- **YouTube** -- gameplay footage for animation and feel references
- **Museum databases** -- for authentic medieval art and architecture
- **Unreal/Unity marketplace screenshots** -- for technical and lighting references
- **Your own photos** -- real medieval buildings, landscapes, castles you've visited
